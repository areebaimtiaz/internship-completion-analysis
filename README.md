
Internship Completion Rate Analysis

 Project Overview

This project analyzes internship performance across multiple departments such as Web Development, AI, Machine Learning, Marketing, and more.

Dataset

The dataset includes:

 Department
 Enrolled, Completed, Dropped
 Duration (weeks)
 Mentor Interaction
 Completion Rate (calculated)

Analysis Performed

 Calculated completion rates using Excel
 Compared department-wise performance
 Analyzed impact of mentor interaction
 Evaluated duration vs completion trends

 Key Insights

 High mentor interaction leads to better completion rates
 Technical fields show higher success rates
 Low interaction results in higher dropout rates
 Longer internships tend to perform better

 Tools Used

 Microsoft Excel

Conclusion

This project demonstrates how simple data analysis techniques can uncover meaningful insights and support decision-making.
